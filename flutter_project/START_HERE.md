## âš ï¸ Common First-Time Issues

### Issue: "Flutter not found"
```bash
# You need to install Flutter first
# See SETUP_GUIDE.md
flutter --version  # If returns error, install Flutter
```

### Issue: "No devices found"
```bash
# Option 1: Open Android emulator before running
# Option 2: Connect physical device
# Option 3: Run iOS simulator on Mac
flutter devices  # List available devices
flutter run -d <device_id>  # Specify device
```

### Issue: "Gradle/Pod errors"
```bash
# Clean everything
flutter clean
# Get dependencies again
flutter pub get
# Try running again
flutter run
```

### Issue: "Permission denied" (macOS/Linux)
```bash
# Make sure you have permissions
chmod +x /path/to/flutter/bin/flutter
flutter run
```

**For more issues**: See [SETUP_GUIDE.md](SETUP_GUIDE.md) Troubleshooting section
